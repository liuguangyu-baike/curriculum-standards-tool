## ⚙️ 需要配置的内容

### 1. 安装 npm 依赖

**当前状态**: 正在安装中...

如果安装失败,请手动执行:
```bash
cd web
npm install nodemailer multer
```

### 2. 配置 SMTP 邮箱 (必需)

需要在 `web/.env` 文件中配置以下参数:

```env
## 邮件配置（用于反馈功能）
SMTP_HOST=smtp.qq.com
SMTP_PORT=587
SMTP_USER=你的QQ邮箱@qq.com
SMTP_PASS=你的授权码
```

**推荐使用 QQ 邮箱**:
1. 登录 QQ 邮箱网页版
2. 设置 -> 账户 -> POP3/IMAP/SMTP服务
3. 开启 IMAP/SMTP 服务
4. 生成授权码(16位字符)
5. 将授权码填入 `SMTP_PASS`

详细配置步骤见: `反馈功能配置说明.md`

### 3. 重启服务器

配置完成后需要重启服务器:
```bash
# 方法1: 使用启动脚本
cd web
./start_server.sh

# 方法2: 手动重启
ps aux | grep server.js
kill <进程ID>
npm run dev
```

## 🚨 注意事项

1. **SMTP 配置是必需的**
   - 未配置时提交会失败
   - 建议使用QQ邮箱(稳定、免费)

2. **授权码不是邮箱密码**
   - QQ邮箱需要生成专用授权码
   - 授权码泄露后需立即重置

3. **网络要求**
   - 服务器需要能访问 SMTP 服务器
   - 端口587(TLS)或465(SSL)需要开放

4. **邮件送达**
   - 首次发送可能进入垃圾邮件
   - 建议将发件地址加入通讯录
