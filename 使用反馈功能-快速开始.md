## ⚡ 快速配置(5分钟)

### 步骤1: 获取 QQ 邮箱授权码

1. 打开 [QQ 邮箱网页版](https://mail.qq.com)
2. 点击右上角 **设置** → **账户**
3. 向下滚动找到 **"POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务"**
4. 开启 **"IMAP/SMTP服务"** 或 **"POP3/SMTP服务"**
5. 点击 **"生成授权码"**，按提示发送短信
6. **复制显示的16位授权码** (类似: `abcd efgh ijkl mnop`)

### 步骤2: 配置 .env 文件

编辑 `web/.env` 文件,找到邮件配置部分,修改为:

```env
## 邮件配置（用于反馈功能）
SMTP_HOST=smtp.qq.com
SMTP_PORT=587
SMTP_USER=你的QQ邮箱@qq.com
SMTP_PASS=刚才复制的16位授权码
```

**示例**:
```env
SMTP_HOST=smtp.qq.com
SMTP_PORT=587
SMTP_USER=1234567890@qq.com
SMTP_PASS=abcdefghijklmnop
```

⚠️ **重要**:
- `SMTP_PASS` 填写的是**授权码**,不是 QQ 密码
- 授权码是16位英文字母,中间没有空格

### 步骤3: 重启服务器

**方法1: 使用启动脚本 (推荐)**
```bash
cd web
./start_server.sh
```

**方法2: 手动重启**
```bash
# 查找进程ID
ps aux | grep server.js

# 停止旧进程 (假设 PID 是 56557)
kill 56557

# 启动新进程
cd web
npm run dev
```

### 步骤4: 测试功能

1. 打开浏览器访问: **http://localhost:8001**
2. 点击顶部导航的 **"使用反馈"** 标签
3. 填写反馈内容 (其他字段可选)
4. 点击 **"提交反馈"**
5. 看到绿色成功提示 ✅
6. 检查邮箱 `liuguangyu@kanyun.com` 是否收到邮件

## 🔍 故障排查

### 问题1: 提交后显示"提交失败"

**可能原因**: SMTP 配置错误

**解决方法**:
1. 检查 `web/.env` 中的配置是否正确
2. 确认 `SMTP_PASS` 是授权码,不是QQ密码
3. 查看服务器日志:
   ```bash
   tail -f /tmp/course-standards-server.log
   ```

### 问题2: 邮件没有收到

**可能原因**:
- 进入了垃圾邮件箱
- SMTP 服务器限流

**解决方法**:
1. 检查垃圾邮件箱
2. 将发件地址加入通讯录
3. 等待几分钟再试 (某些邮箱有延迟)

### 问题3: 图片上传失败

**可能原因**:
- 图片大小超过10MB
- 图片格式不支持

**解决方法**:
- 压缩图片后重试
- 确保图片格式为 PNG/JPG/GIF
