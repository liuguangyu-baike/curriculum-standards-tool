## 📝 新增功能清单

### 1. ✅ 全选功能
**位置**: 课标浏览器 - 结果列表区域

**实现内容**:
- 结果列表表头增加全选复选框
- 结果列表顶部增加"全选"按钮
- 支持三种状态:
  - 未选中 (空复选框)
  - 全选 (勾选复选框)
  - 部分选中 (半选状态,indeterminate)

**验证步骤**:
1. 打开浏览器,输入关键词搜索(如"物质")
2. 点击表头复选框或"全选"按钮
3. 观察所有结果行是否被选中
4. 底部"已选条目"计数是否正确
5. 再次点击取消全选,观察是否正确清空
6. 手动选中部分条目,观察表头复选框是否显示半选状态

**预期效果**:
- 点击全选后,当前页面所有条目被选中
- 跨模式(关键词/匹配)保持全选功能正常
- 半选状态显示正确

## ⚠️ 已知问题与注意事项

### 配置要求
1. **API Key**: 确保 `web/.env` 文件中配置了 `DEEPSEEK_API_KEY`
2. **数据文件**: 确认 `web/data/` 目录包含:
   - dci_data.json
   - pe_data.json
   - cn_compulsory_science_knowledge.json

### 兼容性
- 建议使用 Chrome/Edge/Safari 最新版
- 需要支持 CSS Grid 和 CSS sticky 定位
- 需要支持 ES6 语法

### 可能的问题
1. **全选后切换模式**: 选中状态应该保持,但显示的是新模式的结果
2. **表头对齐**: 如果结果行列宽度异常,表头可能对不齐
3. **英文原文过长**: 可能导致行高变化,需要适当调整样式
