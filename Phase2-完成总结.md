## 🔧 技术实现要点

### 1. 状态管理

```javascript
const appState = {
  currentPage: 'browser',
  browserMode: 'keyword',
  filters: { sources: [], domains: [], grades: [], searchQuery: '' },
  selectedItems: new Set(),
  matchResults: [],
  chatHistory: []
}
```

### 2. 事件流

```
用户操作 → 更新appState → 触发updateResults() → 调用display.js渲染
```

### 3. 模块通信

- **browser.js**: 导出全局状态和函数
- **filters.js**: 更新筛选条件后调用 `updateResults()`
- **display.js**: 提供 `renderResults()` 和 `renderMatchResults()`
- **workbench.js**: 读取 `getSelectedItems()` 构建上下文

### 4. 防抖和性能

```javascript
// 关键词搜索300ms防抖
let debounceTimer;
input.addEventListener('input', (e) => {
  clearTimeout(debounceTimer);
  debounceTimer = setTimeout(() => {
    updateResults();
  }, 300);
});
```
