## 🐛 常见问题排查

### Q1: 页面空白或报错？

**检查**:
1. 打开Console，查看错误信息
2. 确认服务器正在运行（8001端口）
3. 确认 `web/data/` 下有3个JSON文件

**解决**:
```bash
# 重启服务器
cd web && node server.js
```

---

### Q2: 数据没有加载？

**检查**:
1. Console是否显示"数据加载成功"
2. 检查 `standardsData.allStandards` 是否有数据

**调试**:
```javascript
// 在Console执行
console.log(standardsData.allStandards.length);
```

---

### Q3: 智能匹配没反应？

**检查**:
1. Console是否有错误
2. 是否配置了 `web/.env` 文件
3. API Key是否有效

**调试**:
```bash
# 查看.env文件
cat web/.env

# 应包含
DEEPSEEK_API_KEY=sk-...
```

---

### Q4: 筛选不生效？

**检查**:
1. Console输出"筛选变化: ..."
2. 查看 `appState.filters`

**调试**:
```javascript
// 在Console执行
console.log(appState.filters);
```

---

### Q5: 选中状态丢失？

**检查**:
1. 是否刷新了页面（会清空状态）
2. Console是否有JavaScript错误

**调试**:
```javascript
// 查看选中的ID
console.log(Array.from(appState.selectedItems));
```
